¿Qué es el CAF?

El CAF (Correlativo de Autorización de Folios) es un documento electrónico que el SII entrega para autorizar la emisión de documentos tributarios. Es el equivalente moderno del antiguo "timbraje".

Importante

Sin CAF activo no se pueden generar DTEs. El sistema automáticamente asigna folios del CAF disponible al generar documentos.
